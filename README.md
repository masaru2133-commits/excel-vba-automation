# Excel VBA Production Summary Tool

## Overview
Manufacturing data aggregation tool using Excel VBA.
Automatically imports CSV production data and calculates defect rate by product.

## Features
- CSV auto import
- Product-based aggregation
- Automatic defect rate calculation
- Summary sheet generation

## Environment
Excel 365

## Target
Manufacturing process improvement

## Improvement
- Zero division protection implemented
- Data type safety using CDbl conversion
